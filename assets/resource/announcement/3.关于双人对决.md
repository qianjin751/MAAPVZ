# ⚠️ 关于「双人对决」功能的特别声明

感谢各位使用 MaaPVZ 的双人对决自动化功能。为了维持良好的游戏环境与社区氛围，现对该功能的使用做出如下说明：

## 🔍 功能实现原理

- 当前版本的「双人对决」自动化功能**依赖游戏内的特定机制 / 漏洞（Bug）** 实现，并非通过纯粹的图像识别与模拟点击完成。
- 这意味着该功能的稳定性、公平性**不受官方认可**，且存在随时被游戏运营方修复或封禁的风险。

## 🚫 禁止行为 — 请勿“跳脸”

**严禁**在正常玩家（包括但不限于游戏官方社群、贴吧、B站、QQ 群等公开场合）面前：

- 炫耀利用该漏洞取得的胜利或连胜；
- 嘲讽、挑衅正常玩家；
- 公开传播漏洞的具体实现方法或教程。

上述“跳脸”行为会：

- 激怒正常玩家，引发大规模举报；
- 促使游戏运营方加速修复漏洞并追究账号责任；
- 破坏 MaaPVZ 项目的整体声誉。

## ⚖️ 功能存续的底线

本功能仅为**个人学习与研究自动化技术**而保留，**不鼓励、不支持**任何破坏他人游戏体验的行为。

**若出现以下情形，我们将毫不犹豫地从下个版本中移除「双人对决」功能：**

1. 有人在官方游戏社群、论坛或其他公共场合**大肆宣扬该漏洞的存在**，并引起运营方或大量普通玩家的关注；
2. 有人利用该功能进行**商业代打、代练**或**恶意刷分**，造成恶劣影响；
3. 因该功能导致的负面舆论**波及 MaaPVZ 项目本身**（例如官方群内大量用户投诉、开发者被骚扰等）。

## 💬 最后的话

我们制作 MaaPVZ 的初衷是简化日常、解放双手，而不是破坏游戏平衡或惹怒其他玩家。请各位使用者**低调、自律**，珍惜这个来之不易的功能。

> 若您发现有人在公开场合“跳脸”或滥用漏洞，欢迎向项目维护者（QQ群管理 / GitHub Issues）匿名举报，我们会视情况处理。

感谢您的理解与配合。

—— MaaPVZ 开发组
