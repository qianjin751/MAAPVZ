## 🖥️ 模拟器配置要求

1. **分辨率**：必须设置为 **1280×720** 或 **1920×1080**（横屏），否则识别会失败。
2. **推荐模拟器**：MuMu 模拟器、蓝叠模拟器（不推荐雷电模拟器，可能出现图像偏移）。
3. 如果使用多开，请确保 ADB 端口正确（默认 5555, 5557 等）。

## ⚙️ 温馨提示

1. 点击任务旁边的 **齿轮按钮** 可以调整对应任务的详细选项。
2. 请勿滥用自动化功能影响其他玩家正常游戏体验，合理使用。
3. 遇到问题请提供 **日志文件**（`logs/` 目录）到 QQ 群或 GitHub Issues，以便快速修复。
4. Windows 用户请下载 `win-x64` 版本的压缩包；Linux/macOS 用户可自行编译。
