## 🛠️ 如何参与贡献

项目完全开源，欢迎任何人贡献代码、文档或测试反馈。

### 1. 反馈问题

- 前往 [GitHub Issues](https://github.com/Maa-Assistant-PVZ-The-best/MAAPVZ/issues) 提交 bug 或建议。
- 提供详细日志（`logs/` 目录）、截图或录屏，以及您的运行环境（模拟器型号、分辨率、系统版本）。

### 2. 提交代码

- Fork 本仓库 → 创建功能分支 → 开发 → 确保 `pre-commit` 检查通过 → 发起 Pull Request。
- 代码风格遵循 `.pre-commit-config.yaml` 与 `.prettierrc` 配置。
- 主要技术栈：Python 3.12+、.NET 8.0 (Avalonia)、MaaFramework。

### 3. 编写文档

- 帮助完善开发手册、用户使用指南或翻译文档。
- 文档位于仓库 `docs/` 目录或直接编辑 `README.md`。

### 4. 宣传推广（请遵守规则）

- 在不打扰官方玩家社区的前提下，向朋友推荐 MaaPVZ。
- **重要**：请勿在游戏官方社群或公开频道“跳脸”宣传，尤其不要提及利用漏洞的决斗功能。详见 [双人对决特别声明](./DUEL_DISCLAIMER.md)。

## 📢 社区准则

- 尊重每一位贡献者与用户，拒绝嘲讽、引战。
- 禁止利用本工具进行商业代练、破坏游戏平衡或恶意刷分。
- 禁止在游戏官方社群内宣扬漏洞功能，否则该功能可能被移除。
